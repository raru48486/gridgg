# GridGG

A lightweight, high-performance library that adds editing and selection capabilities to HTML table elements. 
Specifically optimized for Japanese inline input (IME) and efficient keyboard workflows.

## Key Features
* 🚀 **Instant Edit**: Transform static HTML tables into interactive, editable components.
* 🖱️ **Range Selection**: Select cell ranges intuitively using mouse dragging or **Shift + Arrow keys**.
* ⌨️ **Full IME Support**: Seamlessly handles Japanese inline input and text composition.
* 🎮 **Enhanced Navigation**: 
    * Use **Arrow keys** to move focus between cells.
    * Use the **Enter key** for rapid data entry and navigation.
* 🛠️ **Customizable**: Minimal footprint and easy to integrate into your existing projects.
